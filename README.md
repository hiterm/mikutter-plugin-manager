## Overview
mikutterのプラグインの有効・無効を切り替えるスクリプト

## Usage
`./mikutter-switch-plugin.rb`
```
command> help
h, help:
  See this help.
e, enable [plugin number]:
  Enable plugin numberd [plugin number].
d, disable [plugin number]:
  Disable plugin numberd [plugin number].
q, quit, exit:
  Quit this program.
```
